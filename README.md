# ConnectPro Backend

Enterprise-Level Professional Networking Platform Backend built using Spring Boot.

## 🚀 Project Overview

ConnectPro is a Professional Networking Platform inspired by LinkedIn.

This project is being developed using enterprise-level architecture and best coding practices.

---

## 🛠 Tech Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Maven
- JWT (Coming Soon)
- IntelliJ IDEA
- Git & GitHub
- Postman
